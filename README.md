# How to Host on GitHub Pages
1. Go to your repository **Settings**.
2. Click on the **Pages** section in the left sidebar.
3. Under **Branch**, select `main` and click **Save**.
4. GitHub will deploy your site. After a minute, refresh the page to see your live link!
